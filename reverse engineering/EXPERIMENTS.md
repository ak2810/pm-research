# EXPERIMENTS

Every modeling experiment is recorded here: hypothesis, method, dataset,
result, conclusion. Append-only.

---

## Format

```
### EXP-NNN — <short title>
**Date**: YYYY-MM-DD
**Phase / Layer**: N
**Hypothesis**: <what we tested>
**Method**: <algorithm, cross-validation scheme, dataset slice>
**Dataset**: <date range, feed, row count>
**Result**: <numeric outcome — R², RMSE, log-likelihood, etc.>
**Conclusion**: <what this tells us; whether hypothesis was confirmed/rejected>
**Commit**: <git hash>
```
